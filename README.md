Dotfiles
========

This repo is my personal dotfiles.

I use [Dotbot](https://github.com/anishathalye/dotbot).

### Installation
Clone to `~/.dotfiles`

```sh
git clone https://github.com/noamokman/dotfiles.git ~/.dotfiles
```

Run the install script

```sh
cd ~/.dotfiles
./install
```
